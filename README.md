# Messenger Bot
Aim of the project:
  * send messege (via `messenger`) if given keyword occurs on given website
  * fully anonymous

# Usage
As a end-user, all you need to do is send request as follows:
`http://example.com keyword#1 keyword#2 keyword#3`
